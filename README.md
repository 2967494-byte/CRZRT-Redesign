# АО «Центр развития закупок РТ» Redesign Concept

This is a conceptual redesign of the Zakupki Tatar website, focusing on a modern, "Apple-style" aesthetic and educational content.

## Features
- **Premium Aesthetics**: Dark mode, glassmorphism, clean typography.
- **Education Focus**: Dedicated sections for courses and training materials.
- **Interactive Elements**:
  - Smooth scroll animations.
  - Hover tilt effects on cards.
  - Custom cursor.
  - Animated statistics counters.

## How to View
Simply open `index.html` (Landing) or `courses.html` (Education Programs) in your web browser.

## Technologies
- HTML5
- CSS3 (Modern features: Flexbox, Grid, Variables, Animations)
- Vanilla JavaScript (IntersectionObserver, DOM manipulation)

## Design Decisions
- **Typography**: Inter (Google Fonts) for clean readability.
- **Color Palette**: Deep blacks, grays, and Apple-blue accent.
- **Layout**: Clear hierarchy with immersive hero section, stats, and course cards.
