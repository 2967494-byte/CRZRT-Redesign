# Пояснительная записка к обновлению ТЗ (версия от 04.05.2026)

В Техническое задание были внесены изменения на основе предложений юридического отдела и правок Заказчика. Основной фокус сделан на детализации результатов работ и разграничении ответственности.

## Основные изменения:

1. **Конкретизация состава работ (Раздел 4.1):** 
   - Зафиксирован перечень из 4-х обязательных посадочных страниц (лендингов).
   - Уточнена навигационная структура (главное меню, хлебные крошки).

2. **Матрица доступа (Раздел 4.3):**
   - Добавлены роли для функционирования Личного кабинета (авторизованные пользователи).
   - Предусмотрены специальные права доступа для управления видеоархивом и обучающим контентом.

3. **Результаты этапов работ (Раздел 6):**
   - Внедрена таблица этапов, где для каждого шага определен физический результат (deliverable), передаваемый Заказчику. Это дает юридическое определение «того, что получаем на выходе».

4. **Разграничение ответственности (Раздел 7.2):**
   - Установлено, что **первичное и последующее наполнение сайта контентом осуществляется Заказчиком**. 
   - Исполнитель отвечает за предоставление инструментов (CMS) и инструкций.

5. **Технические уточнения (Раздел 5):**
   - В список поддерживаемых браузеров включен Яндекс.Браузер.
   - Удалены избыточные требования по интеграции с внешними CRM (Bitrix24) и поддержке SMART-TV.
   - Добавлено требование к административной панели по наличию общего списка всех страниц для массового управления SEO-метаданными.

Данные правки приводят документ в соответствие с текущими договоренностями и минимизируют риски при приемке работ.
